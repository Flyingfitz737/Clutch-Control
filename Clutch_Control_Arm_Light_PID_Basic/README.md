# Clutch Control Arm Light PID Basic

This folder contains a PID implementation with arming light functionality.

## File Structure
- **Clutch_Control_Arm_Light_PID_Basic.ino** - Complete Arduino sketch

## How to Use
1. Open **Clutch_Control_Arm_Light_PID_Basic.ino** in Arduino IDE
2. Install required libraries (check sketch for dependencies)
3. Select your Arduino board
4. Upload to your device

This version includes LED feedback for arming status.
