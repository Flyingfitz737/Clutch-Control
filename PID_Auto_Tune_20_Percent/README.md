# PID Auto Tune 20%

This folder contains a PID auto-tune implementation with 20% tuning parameters.

## File Structure
- **PID_Auto_Tune_20_Percent.ino** - Complete Arduino sketch

## How to Use
1. Open **PID_Auto_Tune_20_Percent.ino** in Arduino IDE
2. Install required libraries (check sketch for dependencies)
3. Select your Arduino board
4. Upload to your device
